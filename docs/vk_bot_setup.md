# 🔵 Настройка VK-бота (Java)

## 1. Получите токен сообщества

- Перейдите в настройки сообщества VK → «Работа с API» → «Ключи доступа».
- Создайте ключ с правами на сообщения.

## 2. Задайте переменные окружения

```bash
export VK_BOT_ENABLED=true
export VK_BOT_TOKEN="<your_vk_token>"
export VK_GROUP_ID=123456
export VK_OPERATOR_CHAT_ID=0
```

## 3. Запуск бота через панель

Откройте «Настройки → Каналы (боты)», выберите канал VK и нажмите «Запустить бота». Панель создаст отдельную SQLite-базу для этого канала и запустит Java-бот в отдельном процессе.

## 4. Проверка

Отправьте сообщение в сообщество VK — заявка должна появиться в панели и в журнале бота.
