# 2026-06-05 18:05:34 - dashboard layout reset and drag fix

## Промпты пользователя

- `при перетаскивании очень часто обе плашки начинают мерцать.`
- `нужен пункт "Вернуть по умолчанию", чтобы вернуть порядок`

## Что изменено

- в `dashboard/index.html` переработан drag-state аналитических панелей: во время перетаскивания исходная плашка временно убирается из потока, а сетка опирается на один placeholder, что заметно снижает мерцание соседних карточек;
- в логике перемещения placeholder добавлены проверки на повторную вставку в ту же точку и исключение placeholder из расчёта drop-target, чтобы убрать лишние reflow при `dragover`;
- над аналитической сеткой добавлена кнопка `Вернуть по умолчанию`, которая снимает все `pin`, восстанавливает исходный порядок панелей и очищает сохранённый layout из `localStorage`.
