## Что сделано

- На странице `dialogs` добавлена реальная пагинация списка:
  - появились кнопки перехода назад/вперёд и индикатор текущей страницы;
  - выбор `Количество диалогов` теперь управляет размером страницы, а не просто обрезает список;
  - при смене фильтров, поиска, SLA-окна и сортировки список корректно возвращается на первую страницу;
  - при открытии конкретного диалога строка автоматически показывается на нужной странице списка.
- В legacy-модалке диалога очищена история от служебного шума:
  - по-прежнему скрываются события оценки/feedback;
  - дополнительно скрываются стартовые клиентские ответы на анкетные вопросы, при этом сохраняется последнее содержательное описание проблемы.
- Кнопки действий legacy-модалки перенесены из нижнего footer в шапку рядом с крестиком:
  - `Взять себе`, `Задача`, `Закрыть`, `Переоткрыть`;
  - для desktop сохранена одна линия, для mobile добавлен перенос без ломки layout.
- В `ai-context/tasks/task-list.md` добавлена запись о выполненной задаче `01-082`.
