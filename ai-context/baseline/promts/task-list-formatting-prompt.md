# Промт: Форматирование task-list

## Назначение

Используй этот промт, когда пользователь просит: `отформатируй задачи`.

## Готовый промт

```text
Отформатируй все неотформатированные задачи в
`ai-context/tasks/task-list.md`.

Правила:

1. Сохрани порядок задач.
2. Всем неотформатированным строкам добавь код и статус.
3. Используй формат:
🔴 [<код>] Короткое описание задачи
4. Если в репозитории нет специальных локальных стандартов, для новых задач используй следующий свободный локальный код в формате `01-XXX`.
5. Если локальные стандарты репозитория требуют использовать Jira key или Bitrix task id как код задачи, используй этот формат.
6. Если во входящей формулировке уже есть валидный Jira/Bitrix код и локальные стандарты репозитория это допускают, сохрани его, а не заменяй локальным кодом.
7. Для задач без детализации ставь статус `🔴`.
8. Уже отформатированные строки не меняй без необходимости.
9. Если пользователь указал другой сырой источник задач, сначала возьми задачи оттуда и перенеси их в `task-list.md`.

После форматирования:
- обнови `task-list.md`;
- кратко перечисли, какие задачи получили новые коды.
```
