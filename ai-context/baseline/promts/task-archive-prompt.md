# Промт: Архивация задач

## Назначение

Используй этот промт, когда пользователь просит: `архивируй задачи`.

## Готовый промт

```text
Архивируй завершенные задачи из `ai-context/tasks/task-list.md`.

1. Найди все строки со статусом `🟢` в `ai-context/tasks/task-list.md`.
2. Убедись, что существуют `ai-context/tasks/task-archive/` и файл
   `ai-context/tasks/task-archive/archive.md`.
3. Для каждой задачи со статусом `🟢` перенеси файл
   `ai-context/tasks/task-details/<код>.md` в
   `ai-context/tasks/task-archive/<код>.md`, если detail-файл существует.
4. Перенеси строки этих задач из `ai-context/tasks/task-list.md` в
   `ai-context/tasks/task-archive/archive.md`, сохранив порядок строк.
5. Удали перенесенные строки со статусом `🟢` из активного
   `ai-context/tasks/task-list.md`.
6. Не трогай задачи с любыми статусами, кроме `🟢`.
7. Не меняй текст задач при переносе в архив.
8. Не создавай отсутствующие detail-файлы заново.
9. Создай отдельный changelog-файл в `ai-context/changelog` по правилу
   «одно изменение = один файл».

Не делай:
- перенос незавершенных задач из рабочего списка;
- изменение формулировок архивируемых задач;
- пропуск отдельного changelog-файла после file changes.
```
