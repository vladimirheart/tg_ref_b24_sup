# Правило 004: Завершение задачи требует алерта и резюме для коммита

## Статус

Действует.

## Правило

После завершения задачи агент должен:

1. показать модальный алерт;
2. подготовить готовое резюме для коммита.

## Зачем это нужно

Во время работы человек часто теряет фокус: переключается между задачами,
чатами, кодом и другими рабочими контурами.

Поэтому после завершения задачи нужно не просто закончить ответ, а явно вернуть
пользователя в рабочий контур:

- модальный алерт помогает сразу заметить, что работа завершена;
- резюме для коммита помогает быстро зафиксировать результат без повторного
  восстановления контекста.

## Обязательный способ показа алерта

На macOS алерт вызывается только через локальный скрипт репозитория:

```bash
./ai-context/baseline/scripts/show-completion-alert.sh "Codex" "Работа завершена."
```

Прямой вызов `osascript` со стороны агента не должен быть основным способом
исполнения правила.

## Обязательный формат резюме

В финальном ответе должна быть секция `Резюме для коммита` с fenced `text`
блоком.

Если изменений в файлах не было, в блоке нужно написать:

```text
коммит не требуется
```

Подробный execution-формат описан в `ai-context/baseline/guides/tasks.md`.
