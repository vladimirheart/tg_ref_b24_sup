# Epics Example

Эта директория хранит только baseline documentation и example-структуру для
`project-manager` режима.

Живой backlog команды должен вестись в
корневом `epics/`, а не здесь.

## Что здесь лежит

- `README.md` - описание общей структуры;
- `_example/` - пример файлового каркаса.

## Что запрещено

- считать `_example/` рабочими эпиками проекта;
- переносить example-файлы поверх существующего `epics/`;
- хранить здесь project-specific backlog.
