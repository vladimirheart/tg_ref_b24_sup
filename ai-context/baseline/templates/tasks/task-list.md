# Task List

Живой рабочий список задач для AI.
Подробные правила task-flow описаны в `ai-context/baseline/guides/tasks.md`.

## Рабочий формат

После команды `отформатируй задачи` каждая строка должна выглядеть так:

```text
🔴 [<код>] Короткое описание задачи
```

Детализация задачи хранится в `task-details/<код>.md`.
