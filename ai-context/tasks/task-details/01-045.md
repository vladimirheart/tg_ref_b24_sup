# Задача [01-045]

## Короткое описание

Импортировать RMS-список из внешнего CSV `C:\Users\SinicinVV\git_h\rms_addresses_full.csv` в проект так, чтобы записи автоматически появлялись на странице мониторинга RMS.

## Что нужно сделать

- Забрать из CSV полезные поля для предзаполнения мониторинга RMS.
- Исключить заведомо закрытые записи `CLOSED*`.
- Перенести очищенный CSV в проект как seed-источник.
- Добавить автозагрузку seed-данных в таблицу `rms_license_monitors` при старте приложения.
- Не создавать дубликаты уже сохранённых RMS-записей.
- Импортировать записи в безопасном режиме:
  - без включения мониторинга;
  - без пароля;
  - с сохранением имени, типа, адреса, порта и логина.

## Связанные файлы

- `C:\Users\SinicinVV\git_h\rms_addresses_full.csv`
- `spring-panel/src/main/resources/rms_addresses_seed.csv`
- `spring-panel/src/main/java/com/example/panel/service/`
- `spring-panel/src/main/resources/static/js/rms-monitoring.js`

## Ограничения

- Паролей в CSV нет, поэтому импортированные записи должны оставаться выключенными до ручного дозаполнения.
- Импорт не должен ломать уже заведённые RMS-записи.
