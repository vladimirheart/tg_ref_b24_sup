# Задача [01-006]

## Короткое описание

Защитить API управления ботами: оставить `start/stop` только на `POST`.

## Цель

Убрать риск неявных изменений состояния через `GET/PUT/PATCH/DELETE` для
управляющих endpoint'ов.

## Что сделать

- Перевести `BotProcessApiController` на `@PostMapping` для `start` и `stop`.
- Проверить, что UI продолжает работать через `fetch(..., { method: 'POST' })`.
- Сохранить текущий `GET` только для `status`.

## Критерии готовности

- `start/stop` не принимают `GET/PUT/PATCH/DELETE`.
- Сборка `spring-panel` успешна.
