# Задача [01-008]

## Короткое описание

Убрать принудительное применение proxy для Telegram вне режима `proxy`.

## Цель

Telegram-бот должен использовать proxy только когда это явно задано в маршруте бота в панели.

## Что сделать

- Ограничить применение proxy в `SupportBot` условием `APP_NETWORK_MODE=proxy`.
- Оставить `NO_PROXY` для остальных режимов (`direct`, `inherit`, `vpn`).
- Сохранить диагностический лог о proxy только для режима `proxy`.

## Критерии готовности

- Бот не берет proxy из окружения, если режим сети не `proxy`.
- Настройка маршрута бота в панели остается источником истины.
