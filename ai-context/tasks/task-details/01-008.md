# Задача [01-008]

## Короткое описание

Убрать UI-шум: debug-логи и `alert` в production-шаблонах.

## Цель

Сделать интерфейс чище для пользователей и снизить риск утечек внутренней
диагностики в проде.

## Что сделать

- Убрать лишние `console.log` из шаблонов dashboard/dialogs.
- Заменить `alert(...)` на единый механизм уведомлений.
- Оставить диагностическое логирование только под флагом debug.

## Критерии готовности

- В production-флоу нет блокирующих `alert`.
- Debug-логи не печатаются без явного debug-флага.
