# Задача [01-119]

## Короткое описание

Убрать hero-плашку «Операционный центр поддержки iguana» со страницы dashboard.

## Цель

Полностью удалить верхний hero-блок dashboard, чтобы страница начиналась сразу с рабочих вкладок и аналитики.

## Контекст

- Пользователь после предыдущей доработки попросил убрать hero-плашку целиком.
- Hero секция и её стили находятся в `spring-panel/src/main/resources/templates/dashboard/index.html`.
- После удаления секции важно не оставлять мёртвый inline CSS, связанный только с hero-блоком.

## Что сделать

- Удалить HTML-секцию hero из шаблона dashboard.
- Удалить связанные CSS-правила и адаптивные остатки, которые использовались только этой плашкой.

## Критерии готовности

- На странице dashboard больше не отображается блок `Операционный центр поддержки iguana`.
- После удаления не остаётся сломанной вёрстки и неиспользуемых hero-стилей в шаблоне.

## Обязательные дополнительные условия

- В финальном ответе подготовить `Резюме для коммита` по формату из `ai-context/baseline/guides/tasks.md`.
- После file changes создать отдельный changelog-файл в `ai-context/changelog` и включить в него пользовательский промт, который привёл к изменению.
