# Задача [01-125]

## Короткое описание

Уплотнить заголовки dashboard и вынести описания в hover-подсказки.

## Цель

Сократить вертикальный размер dashboard и сделать страницу чище, перенеся краткие описания блоков из отдельных строк под заголовками в компактные `i`-подсказки по наведению.

## Контекст

- Пользователь попросил перенести краткое описание в одну строку с названием плашки, а само описание выводить через кнопку `i` при наведении.
- Сейчас на dashboard у большинства panel/chart-блоков под заголовком есть отдельный абзац описания, который визуально раздувает страницу.

## Что сделать

- Добавить на dashboard единый UI-паттерн заголовка с inline `i`-подсказкой.
- Убрать отдельные строки описания под заголовками у основных блоков dashboard и перенести их текст в tooltip.
- Сохранить читаемость и доступность подсказок на hover и focus.

## Критерии готовности

- Название блока и `i` находятся в одной строке.
- Бывшее описание отображается во всплывающей подсказке при наведении или фокусе.
- Высота dashboard сокращается без потери контекста по назначению блоков.

## Обязательные дополнительные условия

- В финальном ответе подготовить `Резюме для коммита` по формату из `ai-context/baseline/guides/tasks.md`.
- После file changes создать отдельный changelog-файл в `ai-context/changelog` и включить в него пользовательский промт, который привёл к изменению.
