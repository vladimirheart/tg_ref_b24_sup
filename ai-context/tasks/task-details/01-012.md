# Задача [01-012]

## Короткое описание

Сделать работу с VLESS/proxy цепочками понятной в UI и добавить быстрый индикатор состояния ботов в левом сайдбаре.

## Цель

- Объяснить пользователю, как добавлять несколько VLESS-профилей и использовать их как failover-цепочку.
- Добавить явную проверку доступности профилей (host/port) прямо из UI настроек.
- Показать мини-статусы текущих ботов в sidebar без открытия модалок.

## Что сделано

- В `settings/index.html`:
  - добавлена кнопка массовой проверки профилей и кнопка проверки в каждой строке;
  - добавлена колонка «Статус» с доступностью и endpoint;
  - добавлены подсказки по мультивыбору (`Ctrl/Cmd + click`) и превью цепочки профилей для project/bots/channel.
- В `SettingsApiController` добавлен endpoint `POST /api/settings/integration-network/profiles/probe`.
- В `IntegrationNetworkService` добавлен публичный probe-механизм `RouteProbeResult` с учетом cooldown/failover состояния.
- В `fragments/navbar.html` и `static/js/sidebar.js` добавлены мини-индикаторы статуса ботов в левом меню «Каналы».
