# Задача [01-029]

## Заголовок

Внедрить intent+slots модель, intent catalog и базовый intent extraction pipeline

## Контекст

Привязка памяти к нормализованному тексту ограничивает обобщение и ухудшает
поиск решений. Нужна semantic-модель с `intent_key` и `slot_signature`.

## Что нужно сделать

- ввести `intent_key`, `slots_json`, `slot_signature` в retrieval/learning pipeline;
- создать таблицу `ai_agent_intent_catalog` и базовый набор intent-ов (10–20);
- реализовать rule-based + dictionary extraction для intent и slot-ов;
- добавить валидацию slot schema на уровне intent catalog;
- расширить memory/search индексы для intent-aware lookup.

## Критерии готовности

- похожие формулировки клиента сводятся к одному intent где это ожидаемо;
- intent extraction работает стабильно на базовых сценариях ресторана;
- slot signature участвует в выборе релевантных решений;
- intent catalog управляется конфигурационно и поддерживает развитие.
