# Задача [01-005]

## Короткое описание

Подкрутить UI страницы настроек на базе референсов 21st.dev.

## Цель

Повысить визуальную читаемость и разделение блоков в настройках, сохранив текущую архитектуру шаблона и стилистическую совместимость проекта.

## Контекст

- Страница `settings/index.html` уже содержит собственный слой CSS для плиток, вкладок и модалок.
- Нужны более явные визуальные «рамки» секций и современный вид в духе карточных dashboard-паттернов.
- Пользователь просил ориентироваться на `https://21st.dev`.

## Что сделать

- Усилить визуальный стиль hero-блока и overview-карточки.
- Сделать вкладки/табы более сегментированными и выразительными.
- Добавить явное разделение контентных блоков внутри tab-pane.
- Сохранить адаптивность для desktop/mobile.

## Критерии готовности

- UI страницы настроек визуально разделен на самостоятельные секции.
- Вкладки и карточки выглядят единообразно и не «слипаются».
- Изменения ограничены шаблоном/стилями, без регрессий функционала.
