# Задача [01-124]

## Короткое описание

Перевести блок команды на dashboard в карточную сетку сотрудников.

## Цель

Сделать секцию `Команда` на dashboard менее шумной и более читабельной за счёт отдельных карточек по каждому сотруднику вместо длинного вертикального списка строк.

## Контекст

- Пользователь попросил сделать каждого сотрудника отдельной ячейкой и выводить все данные внутри этой ячейки.
- Сейчас блок `Время по сотрудникам` рендерится как список строк с вложенными метриками, из-за чего секция читается тяжелее.

## Что сделать

- Переделать CSS блока `staffTimeStats` под карточную сетку.
- Изменить фронтовый рендер `updateStaffTimeStats`, чтобы каждый сотрудник выводился отдельной карточкой.
- Сохранить все текущие показатели внутри карточки, но выстроить их в более компактную иерархию.

## Критерии готовности

- Каждый сотрудник отображается отдельной ячейкой/карточкой.
- Все данные сотрудника находятся внутри его карточки и читаются компактнее, чем в прежнем списке.
- Dashboard продолжает собираться без регрессий.

## Обязательные дополнительные условия

- В финальном ответе подготовить `Резюме для коммита` по формату из `ai-context/baseline/guides/tasks.md`.
- После file changes создать отдельный changelog-файл в `ai-context/changelog` и включить в него пользовательский промт, который привёл к изменению.
