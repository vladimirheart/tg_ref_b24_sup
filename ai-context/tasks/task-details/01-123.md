# Задача [01-123]

## Короткое описание

Уплотнить фильтры dashboard и сжать блок команды.

## Цель

Сделать верхний блок фильтров и секцию команды на dashboard визуально спокойнее: перестроить порядок полей, убрать шумное растягивание выбранных ресторанов, починить сброс активного пресета периода и сократить высоту большого блока команды.

## Контекст

- Пользователь попросил перенести выбор ресторанов под выбор периода и ограничивать список ресторанов выбранным периодом.
- После `Сбросить` период действительно возвращается на текущий месяц, но активная плашка пресета визуально не восстанавливается.
- Большой блок `Команда` на dashboard сейчас занимает слишком много вертикального пространства.

## Что сделать

- Перестроить layout фильтров так, чтобы ресторан шёл под периодом, а список выбранных ресторанов не раздувал весь блок.
- Починить обновление активной плашки пресета при `Сбросить` и синхронизировать доступные рестораны с выбранным периодом.
- Сжать большой блок `Команда` через более компактные отступы, текст и высоту списка.

## Критерии готовности

- Поле ресторанов расположено под полем периода.
- Список ресторанов подстраивается под выбранный период и не растягивает фильтр по высоте.
- После `Сбросить` визуально активен штатный пресет текущего месяца.
- Блок `Команда` занимает заметно меньше места без потери данных.

## Обязательные дополнительные условия

- В финальном ответе подготовить `Резюме для коммита` по формату из `ai-context/baseline/guides/tasks.md`.
- После file changes создать отдельный changelog-файл в `ai-context/changelog` и включить в него пользовательский промт, который привёл к изменению.
