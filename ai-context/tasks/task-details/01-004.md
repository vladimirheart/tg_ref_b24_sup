# Задача [01-004]

## Короткое описание

Восстановить сборку `bot-vk` (совместимость с VK SDK и актуальными сигнатурами
`bot-core`).

## Цель

Сделать модуль VK компилируемым в общем reactor build `java-bot`.

## Что сделать

- Устранить несовместимость импортов/классов VK long poll.
- Исправить конфликт типов (`Integer`/`Long`) в `VkSupportBot`.
- Синхронизировать вызовы `ChatHistoryService` и отправки сообщений.
- Подтвердить `mvn -pl bot-vk -DskipTests compile`.

## Критерии готовности

- `java-bot` компилируется без ошибок для `bot-vk`.
- Нет ошибок компиляции по методам VK SDK.
