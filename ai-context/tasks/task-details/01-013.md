# Задача [01-013]

## Короткое описание

Реализовать мониторинг SSL-сертификатов сайтов: отдельная страница, сроки, цветовые статусы, уведомления в колокольчик за 14 дней, возможность добавить/изменить/удалить сайт и вручную обновить проверки.

## Что сделано

- Добавлена таблица `ssl_certificate_monitors` (Flyway migration `V20`).
- Реализованы `Entity` + `Repository` для мониторинга сертификатов.
- Добавлен сервис проверки SSL сертификатов (`SSLSocket`, чтение `notAfter`, расчёт `days_left`, статус).
- Добавлен scheduler периодической проверки с оповещениями операторов за 14 дней до истечения.
- Добавлены REST API:
  - `GET /api/monitoring/certificates/sites`
  - `POST /api/monitoring/certificates/sites`
  - `PATCH /api/monitoring/certificates/sites/{id}`
  - `DELETE /api/monitoring/certificates/sites/{id}`
  - `POST /api/monitoring/certificates/sites/{id}/refresh`
  - `POST /api/monitoring/certificates/refresh`
- Добавлена отдельная страница мониторинга: `/analytics/certificates`.
- На странице реализованы:
  - список сайтов и сроки действия сертификатов;
  - цветовая индикация статусов (`ok`, `warning`, `critical`, `expired`, `error`);
  - ручное обновление одного/всех сертификатов;
  - добавление, редактирование и удаление сайтов.
- В `analytics/index.html` добавлена кнопка перехода на мониторинг SSL.
