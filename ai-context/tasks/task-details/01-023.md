# Задача [01-023]

## Заголовок

Заменить дополнительные темы на Catppuccin и Amber Minimal, сохранив штатную тему и обновив графики дашборда

## Контекст

В проекте сейчас есть штатная проектная тема и устаревшая дополнительная
палитра `Mono Minimal`. Нужно оставить штатную тему и заменить остальные
варианты на две новые палитры по референсам:

- `Catppuccin`
- `Amber Minimal`

Во втором проходе нужно не только заменить палитры приложения, но и
перестроить сам chart-layer на `dashboard`: обновить цветовую гамму, сделать
theme-aware стилизацию для разных типов графиков и освежить отдельные типы
диаграмм там, где это улучшает читаемость без ломки композиции страницы.

## Что нужно сделать

- оставить текущую штатную проектную тему как основной вариант;
- удалить старую `Mono Minimal`;
- добавить палитры `Catppuccin` и `Amber Minimal` в light/dark совместимом виде;
- обновить переключатели темы в `settings`;
- обновить theme runtime и fallback-логику;
- обновить chart-layer дашборда под новые палитры;
- обеспечить корректную theme-aware перерисовку графиков;
- сохранить композицию страницы и читаемость графиков на `dashboard`.

## Критерии готовности

- доступны три палитры: штатная, `Catppuccin`, `Amber Minimal`;
- `Mono Minimal` больше не предлагается пользователю;
- `theme.js` корректно работает с новыми значениями;
- графики на дашборде визуально согласованы с новыми палитрами;
- разные типы графиков получают осмысленную theme-aware стилизацию;
- при смене темы графики корректно перерисовываются без ручного обновления страницы;
- задача переведена в `🟣` после завершения AI.
