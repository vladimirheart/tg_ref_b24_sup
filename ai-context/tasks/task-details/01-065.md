# Задача [01-065]

Уплотнить левый сайдбар и сократить пустое пространство в навигации.

## Цель

Сделать левый сайдбар визуально компактнее, чтобы он занимал меньше лишнего вертикального пространства, но при этом оставался удобным для чтения и клика.

## Контекст

- Текущий sidebar использует крупные паддинги, большие иконки и заметные вертикальные зазоры между пунктами.
- Пользователь отметил, что в левой навигации слишком много пустого пространства.
- Нужно сохранить общую визуальную стилистику и не ломать collapsed/mobile-поведение.

## Что сделать

- Уменьшить вертикальные отступы контейнера sidebar, header и footer.
- Сделать пункты навигации, их иконки и текстовые блоки компактнее.
- Поджать user-card и нижние action-кнопки sidebar.
- Сохранить читаемость и активные/hover-состояния.

## Критерии готовности

- Sidebar заметно компактнее по высоте без потери функциональности.
- Пункты меню, footer и карточка пользователя занимают меньше места.
- Поведение collapsed sidebar не ломается.

## Обязательные дополнительные условия

- После любых file changes создать отдельный changelog-файл в `ai-context/changelog`.
- В финальном ответе подготовить `Резюме для коммита`.
