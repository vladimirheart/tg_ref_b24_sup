# MCP

Эта директория хранит project-specific контекст по MCP для текущего
репозитория.

## Что сюда класть

- список реально используемых MCP-серверов и коннекторов;
- repo-local ограничения, scope и договоренности по работе с ними;
- краткие playbook-файлы по интеграциям, если они нужны проекту.

## Что сюда не класть

- общие межпроектные правила по MCP из `ai-context/baseline/`;
- токены, пароли, секреты и иные machine-specific доступы;
- generated dumps, кэш и временные файлы.

Секреты для MCP должны жить в `ai-context/parameters/local-machine/`.

Рекомендуется держать здесь обзорный `README.md` и при необходимости
добавлять отдельные файлы по интеграциям, например `github.md`, `figma.md`
или `postgres.md`.
